How to Use Your SSH Key

Quick Start:

  1. Download your SSH public key above
  2. Add it to your ~/.ssh/authorized_keys on the Convergence server
  3. Set proper permissions: chmod 600 ~/.ssh/authorized_keys
  4. Connect using SSH: ssh -i your-private-key user@convergence-server
