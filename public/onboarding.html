1 Account Setup & SSH Keys

üì• Download Your SSH Public Key

üîë Add Your Key to the Server

Once downloaded, add your public key to your authorized_keys file:

Bash

üß™ Test Your Connection

Bash

‚úì Success: You should now be able to connect to the server using SSH.

‚ùì Common Questions

Q: I can't connect via SSH

Check that your public key is in ~/.ssh/authorized_keys with correct permissions (600)

Q: My commits show wrong author

Verify your git config with: git config user.name and git config user.email

Q: How do I update my SSH key?

Download a new key from Account Settings and replace the old one in ~/.ssh/authorized_keys
