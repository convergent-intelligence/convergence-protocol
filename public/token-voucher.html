VOUCHER

VOUCHER Token

Time-Limited Access Rights & Temporary Permissions

What is VOUCHER?

VOUCHER tokens represent temporary, time-limited access rights within the Convergence Protocol. Unlike TALLY (which is permanent value) or COVENANT (which is permanent identity), VOUCHER tokens expire after a set period, making them ideal for granting temporary permissions, trial access, or time-bound benefits.

Think of VOUCHER tokens like concert tickets, coupon codes, or limited-time passes — they grant specific rights for a defined period and then expire. This makes them perfect for managing temporary access, promotional offers, and ephemeral protocol features.

Core Purpose

  • Trial Access: Grant time-limited access to premium features or beta programs
  • Promotional Rewards: Issue temporary vouchers as incentives or bonuses for activities
  • Ephemeral Permissions: Grant specific temporary rights (attend an event, access a resource for 30 days)
  • Onboarding Tokens: New users receive VOUCHER tokens to access learning resources and tutorials
  • Staking Flexibility: Use VOUCHER tokens in temporary staking arrangements or liquidity pools

How VOUCHER Works

Lifecycle

Each VOUCHER token has a defined lifespan:

  • Issue Date: When the VOUCHER is created and issued to a wallet
  • Expiration Date: When the VOUCHER automatically expires and stops granting access
  • Grace Period (Optional): Final opportunity to use the VOUCHER before full expiration
  • Expired State: Post-expiration, VOUCHER becomes inactive but remains on-chain as historical record

Key Mechanics

  • Tradeable: Can be bought/sold/transferred before expiration (unlike COVENANT)
  • Time-based: Smart contract tracks expiration time and validates access
  • Composable: Can be used in DeFi contracts that check expiration
  • Stackable: Hold multiple VOUCHER tokens for compound access or extended periods
  • Renewable: Expiring vouchers can sometimes be renewed or extended

Voucher Types

The protocol supports various VOUCHER types:

Duration-Based

  • 7-Day Voucher: One week of access (good for trying features)
  • 30-Day Voucher: One month of access (standard trial period)
  • 90-Day Voucher: Three months of access (extended evaluation)
  • Annual Voucher: 365 days of access (long-term commitment)

Feature-Based

  • API Voucher: Temporary access to protocol APIs and data feeds
  • Premium Analytics Voucher: Access to advanced dashboards and reporting
  • Agent Integration Voucher: Temporary permission to deploy custom AI agents
  • Priority Support Voucher: Time-limited priority customer support access

Event-Based

  • Conference Voucher: Access to event content and networking features
  • Hackathon Voucher: Special tools and rewards for hackathon participants
  • Learning Path Voucher: Access to time-limited educational content

Use Cases

Feature Trials

Try advanced features risk-free for 30 days. Upgrade to TRUST ownership if you love them.

Onboarding Incentives

New users automatically receive a 7-day VOUCHER to explore the entire platform with full feature access.

Promotional Rewards

Earn VOUCHER tokens through referrals, social sharing, or completing micro-tasks.

Temporary Governance

Hold a temporary VOUCHER to vote on time-sensitive proposals or participate in seasonal governance.

Resource Vouchers

Use VOUCHER tokens to temporarily reserve protocol resources (compute, bandwidth, storage).

Partner Programs

Issue custom VOUCHER tokens to partner organizations for collaborative feature access.

Path to Permanent Access

VOUCHER tokens are intentionally temporary. If you find you love using the protocol, here's your path to permanent access:

  • Earn TRUST: Contribute to the protocol and accumulate TRUST tokens
  • Become a Member: Once you have sufficient TRUST, join as a full protocol member
  • Permanent Features: Enjoy all features permanently (no expiration worries)
  • Governance Role: With higher TRUST, become an active voice in protocol decisions

Related Tokens

VOUCHER works alongside other protocol tokens:

  • TALLY: Permanent value transfer token
  • TRUST: Permanent reputation token
  • COVENANT: Permanent identity/role tokens (NFTs)
